20 git commands and their functions

1. git init: Initializes a new Git repository.

2. git clone [url]: Clones a repository into a new directory.

3. git add [file]: Adds changes in the working directory to the staging area.

4. git commit -m "[message]": Records changes to the repository with a descriptive message.

5. git status: Displays the status of changes as untracked, modified, or staged.

6. git branch: Lists all branches in the repository.

7.  git checkout [branch]: Switches to the specified branch.

8. git merge [branch]: Merges changes from the specified branch into the current branch.

9.  git pull: Fetches changes from a remote repository and merges them.

10. git push [remote] [branch]: Pushes local changes to a remote repository.

11. git log: Displays a log of commits, with commit hashes and messages.

12. git show [commit]: Shows information about a specific commit.

13. git reset [file]: Unstages changes for a file but keeps modifications.

14. git revert [commit]: Reverts a commit by creating a new commit.

15. git rm [file]: Removes a file from both the working directory and staging area.

16. git mv [old] [new]: Renames a file and stages the change.

17. git branch -d [branch]: Deletes a branch.

18. git config --global user.name "[name]": Sets the author name.

19. git config --global user.email "[email]": Sets the author email.

20. git branch -a: Lists all remote and local branches.
